Chrome+新浪微博图床插件=图文并茂的hexo博客

发表于 2018-01-04 | 分类于 Hexo | |

【前面废话，干货转到标题4】
文章中没有几张图片总是显得太单薄，所谓“无图无真相”，很多事情只有用图片才能说得清楚，但hexo上传图片似乎有些麻烦。（懒人真的很无奈）
hexo上传图片的方法有：

1、 本地建立图片文件库，文章编辑时，用markdown语法关联图片的绝对路径。

这种方法，如果文章中的图片很多，那分分钟想要放弃hexo，而且图片会日积月累，如果你够勤奋写了大量博客，那本地图片将会占据很大的空间.如果本地图片误删了，那会有什么后果？

2、 用七牛云存储图片

首先要申七牛账号，每月有10G免费流量（貌似够了，但毕竟有限制，万一你比想象中要勤奋呢），然后各种设置（没用过，不知道）才能上传图片。
优势：方便处理图片，更改图片大小、加水印之类的。提供命令行工具qshell，方便批量上传图片。

3、 用hexo插件上传图片

有支持七牛的插件，貌似很方便，有时间可以试一试。

4、chrome新浪微博图床插件

首先在chrome商店下载新浪微博图床插件（最可靠），也可以自己找资源离线安装chrome插件

然后打开此插件，上传图片，就会得到四种格式的图片地址，我们复制Markdown链接，粘贴到文章中，就会在博客中显示图片了。（马克飞象可以同步预览图片）

可能这种方法不是最好用的，但前期不用折腾，懒人必备！
